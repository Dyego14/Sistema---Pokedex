# Pokédex Interativa - Projeto Front-End

**Descrição:**  
Pokédex interativa dos 151 primeiros Pokémon, desenvolvida com **HTML, CSS e JavaScript puro**.  
O projeto consome dados da [PokéAPI](https://pokeapi.co/) e exibe informações gerais de cada Pokémon ao passar o mouse, como:

- Tipo (em português)
- Altura
- Peso

**Funcionalidades:**  
- Listagem automática de 151 Pokémon  
- Busca dinâmica por nome  
- Hover com informações traduzidas  
- Layout responsivo  

**Tecnologias utilizadas:**  
- HTML5  
- CSS3  
- JavaScript (DOM e fetch API)  
- API REST (PokéAPI)  

**Objetivo:**  
Demonstrar habilidades em front-end, manipulação de DOM, consumo de API externa e criação de interfaces interativas.
